デフォルト設定（作業内容によって変更してください）

  • ●box-sizing: border-box;
    例）幅300px、padding 30px、border 10px

    box-sizing:border-box;

    box-sizing:content-box;

  • ●breakpoint（画面幅による表示の切り替え）
    タブレットサイズ：1024px（$bp-tablet）
    スマホサイズ（iPad未満）：767px（$bp-mobile）
    定義
    @mixin max-screen($break-point) {
    　　　@media screen and (max-width: $break-point) {
    　　　　　@content;
    　　　}
    }
    挿入時のコード
    @include max-screen($break-point) {
    　　　この中にCSSを書く
    }
  • ●font-size: (PC版)16px　(スマホ版)14.4px;
    基準値(PC版)10px、(スマホ版)9pxを1.6remで処理
    font-size系のクラスは全てremを使用
  • ●letter-spacing: 0.05em;
    ●line-height: 1.7;
    ●font-family: ヒラギノ角ゴシック;
  • ●aタグ
    テキスト：デフォルト黒、ホバー時下線追加
    リンク
    画像：transition:0.5s; ホバー時opacity:0.8;
  • ●inputボタン（Safari用）
    Safariはデフォルトでグレーの背景がついてしまうので
    background-color: rgba($color-white,0);
    で削除
